# Task Planner — React App

A simple and efficient Task Planner built with React that helps users manage daily tasks with ease. Users can add, delete, and track tasks in a clean and responsive interface.

# Features

✅ Add new tasks

🗑️ Delete tasks

✔️ Mark tasks as completed

🔄 Real-time UI updates

📱 Responsive design

⚡ Fast and lightweight

# Tech Stack

Frontend: React.js

Styling: Tailwind 

State Management: useState Hook

Build Tool: Vite


## Installation & Setup

1️. Clone the repository    

git clone https://github.com/your-username/task-planner.git

cd task-planner


2️. Install dependencies

npm install


3. Run the development server

npm run dev

App will run at: http://localhost:5173
